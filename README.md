# 🌤️ Moti Sot - Weather App

Aplikacion per parashikimin e motit per qytetet kryesore shqiptare.

## ✨ Funksionaliteti
- Kerko qytetin me emer
- Temperatura, lagështia, era, dukshmeria
- Parashikim 5-ditor
- Ndryshim automatik i sfondit sipas motit
- Mbeshtet: Tirane, Shkoder, Vlore, Durres, Korce

## 🚀 Si ta perdoresh
1. Hap `index.html` ne browser
2. Shkruaj emrin e qytetit dhe shtype Enter ose klikoni 🔍

## 🛠️ Teknologjite
- HTML5, CSS3 (Glassmorphism, Gradients)
- JavaScript (Vanilla)
